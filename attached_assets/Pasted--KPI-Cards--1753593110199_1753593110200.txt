# פרומפט לשינוי יישור טקסט ב-KPI Cards - ממרכז במקום ימין

תודה על העבודה המצוינת! נדרש שינוי עיצובי קטן אך חשוב בריבועי ה-KPI.

## 🎯 המטרה

שינוי יישור הטקסט בכל אחד מארבעת ריבועי ה-KPI בחלק העליון של:
- **מסך לוח הבקרה** (Dashboard)
- **מסך דרישות רכש** (Procurement Requests)

**מה שצריך להשתנות:** יישור הטקסט מימין (text-align: right) למרכז (text-align: center)

## 🔧 שינויים נדרשים ב-CSS

### 1. זיהוי הקלאסים הרלוונטיים

הריבועים שצריכים עדכון הם:
- ריבוע "מדד דיוק AI" (90.8%)
- ריבוע "עליות מחירים" (₪175,000)
- ריבוע "חיסכון מצטבר" (₪991,000)
- ריבוע "סה"כ הוצאה חזויה" (₪7,966,000)

וכן בדף דרישות רכש:
- ריבוע "ערך כולל" (₪7,801,000)
- ריבוע "הושלמו" (1)
- ריבוע "בעיבוד" (10)
- ריבוע "סה"כ דרישות" (11)

### 2. עדכון קובץ CSS

```css
/* עדכון יישור KPI cards למרכז */

/* KPI Cards כלליים */
.kpi-card,
.stat-card,
.metric-card {
  text-align: center !important; /* שינוי מ-right ל-center */
}

/* תוכן KPI Cards */
.kpi-card .kpi-value,
.kpi-card .kpi-label,
.kpi-card .kpi-change,
.stat-card .stat-value,
.stat-card .stat-label,
.metric-card .metric-value,
.metric-card .metric-label {
  text-align: center !important;
  justify-content: center !important;
  align-items: center !important;
}

/* אייקונים ב-KPI Cards */
.kpi-card .kpi-icon,
.stat-card .stat-icon,
.metric-card .metric-icon {
  margin: 0 auto !important; /* מרכז את האייקון */
  display: block !important;
}

/* מספרים גדולים ב-KPI */
.kpi-card h2,
.kpi-card h3,
.stat-card h2,
.stat-card h3,
.metric-card h2,
.metric-card h3 {
  text-align: center !important;
  margin: 0 auto !important;
}

/* תיאורים ב-KPI */
.kpi-card p,
.kpi-card span,
.stat-card p,
.stat-card span,
.metric-card p,
.metric-card span {
  text-align: center !important;
}

/* Flexbox alignment למרכז */
.kpi-card,
.stat-card,
.metric-card {
  display: flex !important;
  flex-direction: column !important;
  justify-content: center !important;
  align-items: center !important;
}

/* Grid של KPI Cards */
.kpi-grid,
.stats-grid,
.metrics-grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  gap: 1rem;
}

/* כל ילד של Grid */
.kpi-grid > *,
.stats-grid > *,
.metrics-grid > * {
  text-align: center !important;
  display: flex !important;
  flex-direction: column !important;
  justify-content: center !important;
  align-items: center !important;
}
```

### 3. עדכון ספציפי לקלאסים קיימים

אם יש קלאסים ספציפיים, הוסף גם:

```css
/* עדכון קלאסים ספציפיים אם קיימים */
.dashboard-kpi,
.dashboard-stat,
.procurement-kpi,
.procurement-stat {
  text-align: center !important;
  justify-content: center !important;
  align-items: center !important;
}

/* עדכון לכל אלמנט עם data-attribute של KPI */
[data-kpi],
[data-stat],
[data-metric] {
  text-align: center !important;
}

/* עדכון לקונטיינרים של KPI */
.kpi-container,
.stat-container,
.metric-container {
  text-align: center !important;
}

.kpi-container > *,
.stat-container > *,
.metric-container > * {
  text-align: center !important;
  margin: 0 auto !important;
}
```

### 4. וידוא תמיכה ב-RTL

```css
/* וידוא שהמרכוז עובד גם ב-RTL */
[dir="rtl"] .kpi-card,
[dir="rtl"] .stat-card,
[dir="rtl"] .metric-card {
  text-align: center !important;
  direction: rtl !important;
}

/* אייקונים ב-RTL */
[dir="rtl"] .kpi-card .kpi-icon,
[dir="rtl"] .stat-card .stat-icon,
[dir="rtl"] .metric-card .metric-icon {
  margin: 0 auto !important;
}
```

## 🔍 בדיקה ואימות

לאחר השינוי, ודא ש:

1. **כל ארבעת הריבועים** בלוח הבקרה מציגים תוכן ממורכז
2. **כל ארבעת הריבועים** בדף דרישות רכש מציגים תוכן ממורכז
3. **האייקונים** ממורכזים בחלק העליון של כל ריבוע
4. **המספרים הגדולים** ממורכזים
5. **התיאורים והתוויות** ממורכזים
6. **אחוזי השינוי** ממורכזים
7. **העיצוב נשמר** זהה למעט יישור הטקסט

## ✅ מה שעובד מצוין (אל תשנה)

- צבעי הרקע של הריבועים
- גבולות וצללים
- גדלים ופרופורציות
- מרווחים בין הריבועים
- צבעי הטקסט
- האייקונים עצמם
- אנימציות (אם יש)
- רספונסיביות

## 🎯 תוצאה מצופה

לאחר השינוי, כל ריבוע KPI יציג:
- אייקון ממורכז בחלק העליון
- מספר גדול ממורכז באמצע
- תיאור ממורכז בחלק התחתון
- אחוז שינוי ממורכז (אם קיים)

הכל יישאר זהה למעט שהטקסט יהיה ממורכז במקום מיושר לימין.

## 📱 תמיכה במובייל

ודא שהשינוי עובד גם במובייל:

```css

  .stat-card,
  .metric-card {

  }
  
  .kpi-grid,
  .stats-grid,
  .metrics-grid {
    grid-template-columns: repeat(2, 1fr) !important;

  }
}

@media (max-width: 480px) {
  .kpi-grid,
  .stats-grid,

    grid-template-columns: 1fr !important;
  }
}
```
